1. Introduction

This client allows the visualisation of time series data provided by OGC Sensor Observation Services.
The development of this client has been performed using the Google Web Toolkit. Thus, this client consists mainly of JavaScript code.
The user behaviour is not analysed by this client. Furthermore, no data are sent to Google.

Legend

The Legend is divided in a button area, underneath legend entries and an overview map.
With the button Add Time Series you can add a time series to the diagram view. The Button Event leads to the event service and the Export button deliver the chosen time series as PDF or CSV file.


A legend entry header consists of label and three buttons with the following function:

  • Changing the diagram style of the time series
  • Extra informations about the sensor
  • Remove the time series out of the diagram view

Under the header are some extra informations and the possibility to jump to the first and latest value of the time series. Besides there are reference values to the chosen time series.


The overview enables a spatially classification of the time series.

Diagram

Within the diagram view the selected time series are displayed as diagrams. Below the main diagram an overview diagram is displayed. By moving and adjusting the grey area, the time period that is displayed in the diagram can be adjusted.

Choose Time Series

This menu allows adding a new time series based on a map view. First, it is necessary to select an according time series service. After this the sensor locations are automatically displayed on the map. After selecting a sensor location on the map, the left window show some informations about the sensor location and the user is able to submit the request for loading the time series.
